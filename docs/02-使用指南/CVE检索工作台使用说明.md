# CVE 检索工作台使用说明

> **如何使用 CVE 补丁检索**

---

## 1. 功能简介

该工作台用于输入一个 `CVE ID`，让系统主动检索补丁、展示运行过程，并提供可复查的补丁证据。

---

## 2. 使用步骤

1. 打开页面 `/cve`
2. 在输入框中输入 CVE，例如 `CVE-2024-3094`
3. 点击“开始检索”
4. 等待运行中状态更新
5. 查看结果摘要与“查看详情”

---

## 3. 页面说明

- **输入框**：仅接受合法 CVE 编号
- **运行状态卡片**：展示当前阶段与最近进展
- **结果摘要**：展示是否找到补丁、主证据与下一步操作
- **详情页入口**：进入完整 trace、patch、diff 查看页

---

## 4. 常见问题

### 为什么一直显示运行中？
- 说明后台还在探索引用页面或下载 patch。
- 若长时间无进展，请查看详情页中的 stop reason 或联系维护者。

### 为什么没有找到补丁？
- 并非所有 CVE 都能直接命中可下载 patch。
- 详情页会展示系统探索过的页面和停止原因。

### 为什么 patch 不能查看 diff？
- 可能该候选未下载成功，或原始内容不可用。

---

## 5. 结果解释

- **已命中补丁**：系统找到了可用 patch/diff，并保留了内容或下载结果。
- **高置信候选**：系统找到了较可信的补丁家族，但不一定都已下载成功。
- **未命中**：系统完成探索，但没有找到满足规则的 patch。
