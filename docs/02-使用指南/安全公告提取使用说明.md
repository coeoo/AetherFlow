# 安全公告提取使用说明

> **如何手动提取一篇安全公告**

---

## 1. 功能简介

安全公告提取支持两种手动输入方式：

- 输入公告 URL
- 直接粘贴公告正文

系统会把输入文档转为结构化情报包。

---

## 2. 使用步骤

### 方式一：URL 提取
1. 打开 `/announcements`
2. 选择“URL 提取”
3. 输入公告地址
4. 点击“开始提取”
5. 查看运行结果摘要
6. 如需发送或复核，进入 `/announcements/runs/{run_id}`

### 方式二：正文提取
1. 选择“正文提取”
2. 输入标题（可选）
3. 粘贴正文内容
4. 点击“开始提取”
5. 查看结构化情报包
6. 在详情页决定是否手动发送

---

## 3. 结果包含什么

- 标题与来源
- 风险级别
- 受影响对象
- IOC 列表
- 修复建议
- 证据片段
- 是否建议推送

---

## 4. 常见问题

### URL 提取失败怎么办？
- 可能页面无法访问、证书异常或正文抽取失败。
- 建议改用正文提取模式。

### 没有 IOC 是否算失败？
- 不算。并非所有公告都包含 IOC。

### 为什么结果字段不完整？
- 原文可能缺少相关信息，系统会先保存可提取部分。
