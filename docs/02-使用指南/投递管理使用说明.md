# 投递管理使用说明

> **如何管理通知目标和查看投递记录**

---

## 1. 功能简介

投递中心用于管理三类通知目标：

- 邮件
- 微信/企业微信
- Webhook

并查看每次发送的记录和失败原因。

---

## 2. 使用步骤

### 新增投递目标
1. 打开 `/deliveries`
2. 点击“新增目标”
3. 选择渠道类型
4. 填写目标信息
5. 保存并测试

### 查看投递记录
1. 打开投递记录标签页
2. 按场景、状态、渠道筛选
3. 查看记录详情
4. 失败时可重试

### 从公告结果页手动发送
1. 打开 `/announcements/runs/{run_id}`
2. 滚动到投递区块
3. 查看系统是否建议发送
4. 选择目标后点击“立即发送”

---

## 3. 注意事项

- 禁用目标不会参与新投递
- 测试发送和正式发送会分开记录
- 敏感配置不会在页面明文展示
- 自动发送与手动发送会共用同一记录中心，但来源动作不同
