# P202 安全公告监控源管理页面设计

> **对应模块：M202 安全公告监控源管理**

---

## 🎯 页面目标

`/announcements/sources` 负责让用户管理首批监控源，并完成以下动作：

1. 查看监控源列表与状态。
2. 新增或编辑一个源。
3. 启停某个源。
4. 立即试跑并跳到批次结果视图。

---

## 🚪 入口与出口

### 入口

- `P201` 点击 `管理监控源`
- `P203` 点击 `返回源配置`
- 直接访问 `/announcements/sources`

### 出口

- 点击 `立即试跑` -> `/announcements?tab=monitoring`
- 点击 `返回手动提取` -> `/announcements`

---

## 🧱 页面布局

```text
Topbar
  -> Page Hero
  -> Source List
  -> Create/Edit Drawer
  -> Inline Run-now Feedback
```

### 区块1：Page Hero

- 页面标题：`监控源管理`
- 简短说明：当前只支持微信文章源、Openwall、NCC

### 区块2：源列表

- 每条源以卡片或行卡片展示：
  - 名称
  - 类型
  - 调度表达式
  - 启用状态
  - 最近一次试跑/抓取摘要

### 区块3：创建/编辑抽屉

- 选择源类型
- 动态切换表单字段
- 可配置：
  - 源名称
  - 调度表达式
  - 抓取配置
  - 通知策略摘要

### 区块4：试跑反馈

- 点击 `立即试跑` 后，在当前行或顶部提示：
  - 已创建批次
  - 跳转查看结果

---

## 🖱️ 关键交互

- 新增与编辑复用同一套表单抽屉。
- 切换源类型时重绘配置字段，但保留已输入的公共字段。
- 启停动作应为行内快捷操作，不进入编辑表单。
- 试跑成功后优先把用户带到批次视图，而不是停留在源列表页猜结果。

---

## 🎭 状态稿

### 默认态

- 展示源列表和 `新增监控源` 按钮。

### 空态

- 无监控源时显示引导卡片：`先创建一个监控源开始持续抓取`。

### 编辑态

- 抽屉打开，列表保持可见但弱化。

### 保存失败态

- 表单内显示字段级和提交级错误。

### 试跑中

- 当前源行显示运行中状态，不冻结整个列表。

### 试跑完成

- 显示 `查看本次批次` 操作入口。

## 📦 页面视图对象

### `MonitorSourceConfigView`

| 字段名 | 类型 | 说明 |
|--------|------|------|
| `source_id` | string | 源 ID |
| `source_type` | string | 源类型 |
| `name` | string | 源名称 |
| `schedule_cron` | string | 调度表达式 |
| `enabled` | boolean | 是否启用 |
| `config_summary` | object | 配置摘要 |
| `delivery_policy_summary` | object | 通知策略摘要 |
| `last_fetch_summary` | object | 最近一次抓取摘要 |

### `MonitorSourceEditorState`

| 字段名 | 类型 | 说明 |
|--------|------|------|
| `mode` | string | `create/edit` |
| `source_type` | string | 当前编辑的源类型 |
| `dirty` | boolean | 是否已修改 |
| `saving` | boolean | 是否正在保存 |
| `validation_errors` | object | 校验错误 |

---

## 🔌 API 与字段映射

| 页面动作/区块 | API | 主要字段 |
|---------------|-----|----------|
| 源列表 | `GET /api/v1/announcements/sources` | `source_id`、`source_type`、`enabled`、最近抓取摘要 |
| 新增源 | `POST /api/v1/announcements/sources` | 创建后的源对象 |
| 更新源 | `PATCH /api/v1/announcements/sources/{source_id}` | 更新后的源对象 |
| 立即试跑 | `POST /api/v1/announcements/sources/{source_id}/run-now` | `fetch_id`、状态摘要 |

---

## 🪞 参考资产与约束

- 监控源管理是平台壳下的场景配置页，不回到老式后台“配置管理中心”。
- 本页可以信息密度略高，但仍以卡片或分组行展现，不鼓励纯管理台表格。
