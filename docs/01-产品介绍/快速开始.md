## 🎯 这是什么

AetherFlow 是一个安全情报处理引擎，以原始安全信号为输入、结构化情报为输出。
承载平台以“平台首页 + 场景工作台”的形态承载首批场景：

- `CVE 补丁检索`：给定一个 CVE，主动检索补丁与证据。
- `安全公告提取`：对安全公告做结构化提取，并支持持续监控。

## 🧭 产品入口

### 平台首页
- 查看两个场景入口
- 查看最近任务与投递记录
- 查看系统状态摘要

### CVE 工作台
- 输入 CVE 编号
- 发起 graph run
- 查看运行状态、证据链、补丁和 diff

### 安全公告工作台
- 手动输入公告 URL 或正文
- 查看结构化情报包
- 配置监控源、调度和通知目标
- 查看监控批次与单文档提取结果

### 投递中心
- 管理邮件、微信/企业微信、Webhook 目标
- 查看投递记录与失败重试情况

### 系统工具页
- 查看平台任务与 attempt 轨迹
- 查看 API / Worker / Scheduler / Database 状态
