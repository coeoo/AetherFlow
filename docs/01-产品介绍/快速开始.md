## 🎯 这是什么

AetherFlow 是一个安全情报处理引擎。
平台以“首页 + 场景工作台 + 详情页”的产品形态承载首批两个场景：

- `CVE Patch Agent`：输入一个 CVE，主动执行多源 seed 聚合、
  多跳页面探索、patch 收敛、下载校验与证据回放。
- `安全公告提取`：对安全公告做结构化提取、监控和投递。

## 🧭 产品入口

### 平台首页

- 查看平台定位与场景入口
- 查看最近任务、最近投递和系统状态摘要
- 从首页直接跳入 Patch 搜索或公告工作台

### CVE Patch 搜索工作台

- 输入 CVE 编号
- 发起一次 Patch 搜索
- 查看运行状态、预算推进与搜索摘要
- 进入详情页复核搜索图、决策历史、patch 收敛与 diff

### 安全公告工作台

- 手动输入公告 URL 或正文
- 查看结构化情报包
- 跳转监控源管理
- 查看监控批次、批次详情与关联公告 run

### 投递中心

- 管理邮件、企业微信、Webhook 目标
- 测试发送目标并查看投递记录
- 执行立即发送、计划发送与失败重试

### 系统工具页

- 查看平台任务与 attempt 轨迹
- 查看 API、Worker、Scheduler、Database 状态
